# Flight API

A microservices-based flight booking API system.

## Services
- Flight Search
- Hotel Search
- Bus Search
- Car Rental
- Event Booking
- Holiday Packages
- Insurance
- Train Booking

## Getting Started
1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run the service: `uvicorn main:app --reload`
